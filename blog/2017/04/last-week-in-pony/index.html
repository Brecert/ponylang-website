Last Week in Pony - April 3, 2017

Wednesday, April 05, 2017 | Posted in Last Week in Pony

Last Week In Pony is a weekly blog post to catch you up on the latest news for the Pony programming language. To learn more about Pony check out our website, our Twitter account @ponylang, our users’ mailing list or join us on IRC.

Got something you think should be featured? Drop us an email at last.week.in.pony@gmail.com.

Items of note

Releases galore!

There were several “point” releases last week. For a list of full releases see the releases page. If you are running on Windows, it is recommended that you update to 0.12.3 as soon as possible. For those on developing on other platforms, upgrading to at least 0.12.1 as soon as possible is recommended.

Update recommended for Windows users.
Version 0.12.2
Update recommended for Windows users.
Version 0.12.1
